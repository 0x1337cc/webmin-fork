Cluster kopieeren Bestanden
Met deze module kunt u een geplande overdracht maken om bestanden van een master server te kopieeren naar andere servers in een Webmin cluster. Dit kan erg handig zijn voor bestanden zoals /etc/hosts, httpd.conf en andere bestanden waar geen netwerk protocol voor beschikbaar is zoals NIS of LDAP.

Op de module hoofdpagina word een lijst getoond van reeds geplande kopieën, en er bevind zich een link om een nieuwe aan te maken. Voor iedere kopie kunt u een bron bestand definiëren, een bestemming directory, een bestemming server en de tijden waarop het uitgevoerd moet worden. De eerste stap is een doel bestemming aanmaken in de Webmin Servers Index module met een gebruikersnaam en wachtwoord.
