Deze module kan ook worden gebruikt om knoppen aan te maken voor files die u regelmatig regelmatig moet bewerken. Het is ook handig omdat u op deze manier niet vertrouwde gebruikers hiermee een knop kunt geven waarmee zij alleen bepaalde files kunnen bewerken.

Net zoals bij de opdracht knoppen zal de hoofdpagina een knop tonen voor elke file bewerker. Om een reeds gedefinieerde bewerker te wijzigen klikt u op de Bewerk File Bewerker knop. Om een nieuwe bewerking aan te maken klikt u op de Maak een nieuwe file bewerker knop onderaan deze pagina.
